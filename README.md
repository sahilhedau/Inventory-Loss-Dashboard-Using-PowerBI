# 📊 Inventory Loss Analysis Dashboard (Power BI)

## 📌 Overview
This project presents an interactive Power BI dashboard to analyze inventory loss across three categories: **Expired, Damaged, and Near Expiry products**. It helps identify high-risk items, warehouses, and suppliers, enabling better inventory control and reduced wastage.

---

## 🎯 Objectives
- Analyze inventory loss patterns  
- Identify high-risk products and locations  
- Monitor expiry and damage trends  
- Support data-driven decision making  

---

## 📁 Dashboard Pages

### 🔴 Expired Analysis
- Total expired products and expiry rate  
- Average days to expiry  
- Monthly trend analysis  
- Distribution by warehouse, product, and supplier  

### 🟠 Damaged Analysis
- Total damaged products and damage percentage  
- Category-wise and warehouse-wise distribution  
- Product-level damage insights  

### 🟡 Near Expiry Analysis
- Total near expiry products and percentage  
- Ageing bucket analysis  
- Dead stock identification  
- Supplier and warehouse contribution  

---

## 📊 Features
- Interactive filters (Product, Warehouse, Supplier, Category)  
- KPI cards for quick insights  
- Multi-dimensional analysis  
- Clean and intuitive dashboard design  

---

## 🛠 Tools & Technologies
- Power BI  
- DAX (Data Analysis Expressions)  
- Data Cleaning & Transformation  

---

## 📌 Conclusion
This dashboard provides a comprehensive view of inventory loss, helping businesses track critical issues and make informed decisions to improve operational efficiency.
